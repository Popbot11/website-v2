the goal for this project is to have a modular effect plugin that specalizes on instantiating many of the same effect in parallel. Things like modal filters and freeverb early reflections work by duplicating the same simple filter many times in parallel with slight diferences in parameters. Most of my fav sounds i've made were done by abusing properties of these types of systems, so i wanted to generalize this behaviour in a modular context. currently i'm still working on writing the basic modular implementation, so i'm gonna hold off on putting more info on this page until i have something that other people can mess with.
